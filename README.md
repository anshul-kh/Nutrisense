<h1>NutriSense</h1>
<p>An Mobile App , that uses Gemini AI API to suggest whether you should eat a packed item or not based on one's medical history.</p>

<h2>How To Run</h2>

[Step 1]:
``` 
git clone <repo-url>  
 ```

[Step 2]:
``` 
npm install  
 ```

[Step 3]:
``` 
npm run start
 ```

[Step 4]:
Scan The URL in ```Expo Go``` App in android.

[Step 5]:
Do the testing..
